sailplayAction
directive in module sailplay.actions

Description

Simple directive for parsing dom element as SailPlay action.

Usage

as attribute
<ANY sailplay-action
     action="{object}"
     styles="{string}"
     text="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
actionobject

A SailPlay action object, received from api.

stylesstring

Not required attribute, used for custom styling iframe buttons.

textstring

Not required attribute, used for custom text in iframe buttons.
