# Oldi loyalty app

## Install
First, install Webpack globally
```
npm install -g webpack 
```
Then install local dependencies
```
npm install
```

## Demo

dev-thanks.html

## Dev
```
npm install -g webpack-dev-server
webpack-dev-server
```

Open index.html in browser

## Production
```
npm start
```
